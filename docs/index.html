Featured Experiences
The mirrors

Two installations where the AI learns to reflect. The technology disappears; what remains is the encounter.

Echoes

Audio Mirror

An oracle that speaks in your own voice

You speak. The installation listens — not just to your words, but to the patterns in how you say them. It asks questions that reach toward vulnerability. Then it synthesizes what it observed and speaks back to you in your own cloned voice, naming things you didn’t know you were showing.

MLX · Whisper · Afterwords TTS · Emotion detection
Reflections

Mirror of Truth

A mysterious entity that observes without judgment

A camera watches your face. A prosody analyzer reads the shape of your speech. An AI entity forms and reforms, tracking your emotional trajectory across the interaction. The mirror never lies — it only shows you what you are already doing.

Webcam · Emotion detection · Prosody analysis · Voice synthesis

Quick start
# Clone and install
git clone https://github.com/adrianwedd/lunar_tools_prototypes.git
cd lunar_tools_prototypes
python3 -m venv env && source env/bin/activate
pip install -r requirements.txt && pip install .

# Copy and configure API keys
cp .env.example .env

# Run any installation
python lunar_tools_demo.py --demo audio-mirror
python lunar_tools_demo.py --demo ai-mirror-of-truthclick to copy
